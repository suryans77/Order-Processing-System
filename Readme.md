# Order Service System

An event-driven order processing system built with Spring Boot that models real-world
e-commerce workflows with asynchronous processing and failure handling.

## Features
- Order lifecycle management
- Idempotent order creation
- Event-driven architecture
- Payment & inventory simulation
- Failure handling and compensation logic

## Tech Stack
- Java 21
- Spring Boot 3.4
- Spring Web
- Spring Data JPA
- Spring Security
- H2 (dev), PostgreSQL (prod-ready)
- Maven
